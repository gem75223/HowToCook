# 如何決策喫什麼

如何決策喫什麼也是我做菜之前一大難題。所以只能用數學描述一下了。

## 計算方法

### 計算葷菜和素菜數量

* 菜的數量 = 人數 + 1。
* 葷菜比素菜多一個，或一樣多即可。

由此得到葷菜數量和素菜數量，再在上一步的菜譜中選擇即可。

#### 形式語言描述

當 有人數 `N` 時，
設 `素菜數` 爲 `a`, `葷菜數`爲 `b`。
`N`, `a`, `b`均爲整數。

此時有下列不等式組：

* a + b = N + 1
* a ≤ b ≤ a+1

解得

```javascript
const a = Math.floor((N+1)/2);
const b = Math.ceil((N+1)/2);
```

### 菜的選擇

* 如果人數超過 8 人，考慮在葷菜中增加魚類葷菜。
* 如果有小孩，考慮增加有甜味的菜。
* 考慮增加特色菜、拿手菜。
* 注意決策葷菜時不要全部使用同一種動物的肉。考慮順序爲：`豬肉`、`雞肉`、`牛肉`、`羊肉`、`鴨肉`、`魚肉`。
* 不要選擇奇奇怪怪的動物做葷菜。
