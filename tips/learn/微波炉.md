# 使用微波爐

## 什麼是微波爐

微波爐是 1945 年由 [珀西·勒巴朗·斯賓塞](https://en.wikipedia.org/wiki/Percy_Spencer) 發明的。

他在擔任雷達系統工程師時，由於發現雷達一開啓他口袋裏的巧克力棒就開始融化，從而產生構想併發明的。

### 工作方式

微波時通過磁控管制造的頻率 24.5 億赫茲的電磁波，這個頻率會使水和油的分子振動併發熱。

## 流程

微波爐在很多烹飪任務中效果相當出色。

強火適用於：

* [烹煮] 烹煮蔬菜
* [烹煮] 軟化含水率高的硬質蔬菜（如馬鈴薯、洋蔥和朝鮮薊）
* [膨化] 爆點心，如泡芙、印度帕帕達姆薄脆餅、爆米花

中火適用於：

* [烹煮] 海鮮 （例如 [微波蔥姜黑鱈魚](../../dishes/aquatic/微波蔥姜黑鱈魚.md)）
* [烹煮] 軟化肉類
* [脫水] 乾燥蔬果皮
* [脫水] 製作肉乾
* [炸] 炸脆香料植物
* [炸] 軟化葉類蔬菜
* [加熱] 加熱剩菜

弱火用於：

* [解凍] 解凍食物
* [解凍] 融化黃油和巧克力

## 注意事項

* 微波爐不應該被用於加熱水，這可能會致其暴沸，對您的人身安全造成一定影響
* 絕對不要使用微波爐加熱雞蛋！除非它是**去殼**的**生**雞蛋，並且需要戳破蛋黃。因爲加熱會使雞蛋內部的氣體膨脹，導致爆炸。
* 不管您是出於什麼目的，即使您很好奇，也不應該使用微波爐加熱完整的水果（如 葡萄、藍莓、聖女果），這可能會使其爆炸
* 由於微波的波長爲 12.2 cm，因此微波爐加熱小物體的速度要比大物體慢。因此如果是很小的食材，建議聚集在一起進行加熱。
* 微波僅能深入食物幾釐米，因此有時候外部很燙了，內部可能還是冰涼的。解決辦法是將食材加上少量液體放進密封袋，或放入碗中後蒙上保鮮膜，讓容器內產生足夠的蒸汽來彌補微波爐容易烹飪不均的缺點。
    * tips：打開密封袋時，當心蒸汽噴出
* 微波只能加熱水和油等含有液體分子的物體，因此保鮮膜和密封袋都不會被微波加熱。
* 金屬能夠反射微波而不會被加熱，請避免使用含有金邊、金屬花紋的容器，避免受熱不均導致容器破裂。請務必使用瓷、玻璃容器或微波爐專用烤盤。
