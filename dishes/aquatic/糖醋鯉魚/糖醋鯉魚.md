# 糖醋鯉魚的做法

預估烹飪難度：★★★★

## 必備原料和工具

- 鯉魚
- 番茄醬
- 白糖
- 白醋
- 澱粉
- 鹽
- 蔥
- 姜
- 料酒
- 香菜一顆
- 盆（兩個）
- 菜刀一個
- 笊籬一個、鍋鏟一個

## 計算

注意，該菜只有 3 人以上版本（過大或太小的魚都是不合適的），所以不需要公式計算，特別適閤家庭聚餐時食用，如年夜飯

- 鯉魚 = 約 3 斤
- 清水 = 50g
- 番茄醬 = 40g
- 白糖 = 20g
- 白醋 = 10g
- 澱粉 = 10g
- 鹽 = 30g
- 大蔥 = 30g（約半顆）
- 姜 = 30g
- 料酒 = 25g

## 操作

- 將魚清洗乾淨，確保無魚鱗等異物
- 將魚頭朝左，魚肚朝下，右手持刀。刀豎直切下 1cm，按緊魚身往左片 3-4cm，再將魚片中間輕輕劃一刀
- 將魚放進盆裡，然後將大薑切片，大蔥切段（隨便切切就行了，主要是需要去腥味）
- 用吃奶的力氣將大蔥大姜裡的汁水擠到盆中
- 加入 20g 鹽，25g 料酒，然後給鯉魚搓個澡，塗抹均勻
- ![醃製](./醃製.jpg)

    （醃魚時間越長，魚腥味就越小，推薦醃 30 分鐘以上）

- 找個乾淨的盆，加入 100g 麵粉、200g 澱粉、180g 水、5g 鹽，用手將其攪拌均勻，麵糊此時粘稠呈可拉絲狀態，然後打入一個雞蛋，再次攪勻
- 等待 30 分鐘
- 將魚放在案板上，用乾毛巾將魚身上的水擦乾（這樣可以更好的掛糊）
- 將盆沖洗乾淨，用乾毛巾擦乾
- 起鍋燒油，加入約 1L 的油，將油溫燒至 7 成熱，約 200-240 度
- 捏起魚的尾巴，將魚頭沉入鍋底，用勺子往魚的身上淋熱油，待麵糊成型後，將魚慢慢放入鍋中，拿鍋鏟輕輕剷起魚的頭部，然後墊上笊籬。防止底部炸糊。
- 準備一個盛魚的盤子，放在鍋的旁邊。
- 用鍋鏟從魚身處輕輕鏟入，兩個工具配合魚翻個身。再炸兩分鐘，還是同樣的方式（笊籬託著魚頭，鍋鏟託著魚身，將魚盛入盤中）
- 將鍋中的油倒入擦乾的盆中，放置一邊，然後將鍋刷乾淨
- 將 50g 清水、40g 番茄醬、20g 白糖、10g 白醋放入小碗中，攪拌均勻
- 再準備一個小碗加入 10g 澱粉、10g 水，攪拌成水澱粉
- 開大火將鍋燒熱，然後倒入之前準備的料汁，大火燒開，轉小火
- 加入調好的水澱粉，邊倒邊攪拌，然後 20 秒後關火
- 將熬好的糖醋汁用勺子均勻地澆在魚身上，可以加點香菜或蔥花點綴，糖醋鯉魚就做好了
