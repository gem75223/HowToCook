# 黃燜雞的做法

黃燜雞是一道十分下飯的美食，食材平平無奇又十分容易烹製，一學就會。

預估烹飪難度：★★★

## 必備原料和工具

- 雞腿
- 香菇（幹香菇最佳）
- 青椒
- 生薑片
- 幹辣椒
- 鹽
- 料酒
- 白胡椒粉
- 白糖
- 醬油
- 味精
- 土豆（可選）

## 計算

- 雞腿 = 兩隻
- 香菇（幹香菇最佳）= 5 朵
- 青椒 = 兩個
- 生薑片 = 兩片
- 幹辣椒 = 5，6 個
- 鹽 = 10g
- 料酒 = 10ml
- 白胡椒粉 = 5g
- 白糖 = 5g
- 醬油 = 5ml
- 土豆 = 一個（可選，可使湯汁更粘稠）

## 操作

### 備菜階段

- 雞腿洗淨，剁成**4cm**大小的塊
- 生薑切片、幹辣椒切成**小圈**
- 香菇切片，青椒切成細長的**馬蹄狀**，若為幹香菇，洗淨灰塵後泡一晚上並留香菇水備用
- 若有土豆，切為與雞肉大小類似的**滾刀塊**

### 炒制階段

- 炒糖色：鍋裡倒入底油，冷油時放入白糖（**有一定難度，新手可跳至雞肉炒制並使用老抽替代**）
- 小火慢慢加熱，待油溫逐漸升高，白糖開始融化並變成較深的棕色（期間要不斷攪拌，防止糊鍋）
- 迅速倒入雞塊，轉大火，快速翻炒！烹入料酒，繼續翻炒片刻
- 將生薑片和幹辣椒倒入
- 放入醬油，炒勻
- 倒入香菇水或清水，以能淹住雞肉為準
- 倒入香菇片，白胡椒粉，鹽，土豆
- 翻炒均勻後，蓋上鍋蓋燜煮，轉中小火**15——20分鐘**，有條件可以轉至砂鍋
- 待雞肉軟爛，湯汁濃稠後（湯汁不要收的太乾），最後放入青椒
- 放入味精，兜炒均勻後，關火！青椒基本斷生即可，不要炒時間久了
