# 土豆燉排骨的做法

預估烹飪難度：★★★

## 必備原料和工具

- 肋排
- 土豆
- 姜
- 小蔥
- 料酒
- 白糖
- 幹辣椒
- 八角
- 花椒
- 桂皮
- 生抽
- 老抽
- 蠔油
- 黃豆醬

## 計算

- 肋排 = 750g
- 土豆 = 300g
- 姜 = 30g
- 小蔥 = 25g
- 料酒 = 25g
- 白糖 = 10g
- 幹辣椒 = 5g
- 八角 = 5g
- 花椒 = 5g
- 桂皮 = 5g
- 生抽 = 10g
- 老抽 = 5g
- 蠔油 = 5g
- 黃豆醬 = 5g

## 操作

- 土豆兩個滾刀切片，薑片切片
- 排骨 750g 冷水下鍋，加入薑片、蔥段、料酒焯水 2 分鐘，焯幹水後撈出清洗乾淨（一定要用熱水，不能用冷水）
- 熱鍋涼油，將白糖倒入鍋中，翻炒至融化為焦糖色
- 加入排骨煎至兩面金黃，讓排骨裹滿焦糖
- 加入幹辣椒、八角、花椒、桂皮、薑片（建議買超市的香料包）、10ml 生抽、5ml 老抽、5ml 料酒、5ml 蠔油、5ml 黃豆醬
- 大火翻炒均勻後加入 700ml 開水，大火燒開後轉小火燜煮 1 小時
- 最後加入土豆煮 10 分鐘就可以出鍋啦（喜歡吃青紅椒的也可以按自己喜好加入）
