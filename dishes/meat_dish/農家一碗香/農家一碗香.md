# 農家一碗香的做法

![農家一碗香](./農家一碗香成品.jpg)

農家一碗香，是一道地道的湖南菜，裡面主要食材有青椒、雞蛋和豬肉。味道鹹香下飯，而且這道菜烹飪簡單，不需要特別的處理。

農家一碗香是一道中等難度的菜品。預計備菜 7 分鐘，烹飪 10 分鐘，總計 17 分鐘。

預估烹飪難度：★★★

## 必備原料和工具

+ 豬肉（推薦五花肉）
+ 青椒
+ 蒜片
+ 豆瓣醬
+ 醬油
+ 小米椒
+ 白糖
+ 薑片

## 計算

每次製作前需要確定計劃做幾份。這裡一份夠 1~2 個人吃。

+ 豬肉 250g
+ 青椒 3 個
+ 蒜片 2 片
+ 豆瓣醬 10mg
+ 小米椒 1 個
+ 白糖 5mg
+ 醬油 15ml
+ 姜 2 片

## 操作

+ 備菜階段：將豬肉切片，最好把肥瘦分開放。同時要把青椒和小米辣切成段，蒜片用刀背拍成末，姜切成絲。雞蛋打到小碗中，用筷子打散。
+ 備好菜，就可正式煮菜了。先將油倒入鍋中，鍋中開小火，油熱後將蛋液倒入鍋中，將雞蛋炒散，炒至斷生即可，放回小碗中備用。
+ 將鍋中繼續加一點油，開小火，鍋熱後將之前切的肥豬肉倒入鍋中逼出豬油。
+ 肥肉出現金黃色的時候，加大火力到中火，將瘦肉一起放入鍋中翻炒。
+ 瘦肉全部炒至變色的時候加入備好的薑絲、蒜末和豆瓣醬翻炒均勻給豬肉上色。
+ 放入青紅椒、炒至斷生的雞蛋後加入醬油和白糖，繼續將它們翻炒到青椒微微斷生，保持青椒清脆口感。
+ 出鍋上菜！

## 附加內容

+ 如果第一次炒雞蛋的時候放的油太多，炒豬肉的時候可以視情況不加油。
